## Installation:
1. Clone this repo:
```
$ git clone https://github.com/misakimichy/react-tic-tac-toe.git
```

2. Install npm:

```
$ npm install
```

3. Run the project:
```
$ npm run start 
```

4. Open up localhost:8080
